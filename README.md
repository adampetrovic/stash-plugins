# Stash Plugins

Custom plugins for [Stash](https://stashapp.cc).

## Source Index URL

Add this URL in **Settings → Plugins → Available Plugins**:

```
https://adampetrovic.github.io/stash-plugins/main/index.yml
```

## Plugins

| Plugin | Description |
|--------|-------------|
| [HEIC to JPEG Converter](plugins/heic-converter/) | Converts HEIC/HEIF images to full quality JPEG using ImageMagick |
